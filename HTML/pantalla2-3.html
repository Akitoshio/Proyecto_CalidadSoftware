    Herramientas de calidad

    El SQA se apoya en herramientas tecnológicas que facilitan la prevención y corrección de errores:

    Contenido

    Pruebas unitarias:

    JUnit (Java), NUnit (.NET). Permiten verificar automáticamente funcionalidades pequeñas del código.

    Pruebas de interfaz:

    Selenium, Cypress. Automatizan pruebas de aplicaciones web simulando la interacción de un usuario.

    Analisis estatico de codigo:

    SonarQube, PMD. Detectan vulnerabilidades, duplicación de código o malas prácticas.

    Gestion agil:

    Jira, Trello, Asana. Organizan tareas y flujos de trabajo bajo enfoques ágiles.

    Integracion y entrega continua (CI/CD):

    Jenkins, GitHub Actions, GitLab CI. Permiten integrar cambios de código automáticamente y lanzar pruebas en cada entrega. Estas herramientas complementan la cultura de calidad, permitiendo que los equipos automatizen procesos y reduzcan errores humanos.
